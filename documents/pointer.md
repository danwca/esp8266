# 指针
C语言的一个特点，就是指针概念；那么，什么是指针？指针是一种基本数据类型，其字节大小由CPU决定，通常可以考虑为两个字节大小；指针与其他类型一起使用，对指针所指向空间的操作进行再次定义。与C语言紧密相关的C语言特性，关键字，声明与定义。在C语言代码中只能使用关键字，C语言定义了大概30多个关节字，其他自定义关键字通过声明完成；关于声明与定义，参见声明与定义一节；

## 声明指针变量
所有指针变量通过两个关键字进行声明，普通类型关键字和符号（*）。示例代码：
・・・
int * iptr;	// 声明指针变量：iptr；所指向空间类型为int；
void (*funcptr)();	// 声明指针变量：funcptr；所指向控制类型为函数代码；
・・・

## 指针在功能码中的使用
此系统中仅有一个数据交换空间（fn_data），其他位置仅仅保存数据空间相关的指针，实现对数据交换控制的操作；


## 声明与定义
声明，是通过已有关键字声明新关键字；定义，是通过逻辑代码定义关键字；
对同一关键字可以进行多次同样的声明；对同一关键字只能进行一次定义；
・・・
//各种声明，参考所有h文件；以下是简单的示例
int x;	// 声明变量：x
void func(int i, char c);	// 声明行数：func
class cls;	// 声明类：cls

class cls	// 声明类及结构
{
	void func1();
}

//各种定义，参考所有c文件；以下是简单的示例
void func(int i, char c)
{
	// any code
}

void cls::func1()
{
	// any code
}
・・・
